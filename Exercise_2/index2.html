How to navigate the website:

Since most of the links in this website are just for show, here is a quick guide on how to access functioning part of the website.

  1. The 2 courses that you can use are Python and C++. Of those courses only the first subcategory works (Introduction to Python and Introduction to C++).

    • Path to "Introduction to Python": Categories>Introduction to Python.
    • Path to "Introduction to C++": Categories>Introduction to C++.

  2. The Python course has includes all the basic functionality for subcategory. In there you will find a working Lecture and Book along with other suggestions that are non-functioning. You will find the functioning parts in the first item of their respective lists for Lectures and Books.

    • Path to Lecture: Categories>Introduction to Python>Introduction to basic concepts.
    • Path to Book: Categories>Introduction to C++>Thinkpython2.
